Overview
This repository contains two versions of the monograph "Out-of-Time-Ordered Echoes in a Quantum-Fractal Tensor Network Framework" by Justin Candler (Nous Enterprises LLC, dated January 16, 2026). The work integrates Google's 2025 OTOC(2) Quantum Echoes experiments with the Quantum-Fractal Tensor Network (QFTN) framework, exploring fractal entropy dissipation, pointer state geometry, spectral dimensions, and more.

V1 (Out_of_Time_Ordered_EchoesV1.pdf): Original version with detailed abstract and core content spanning 30 pages.
V2 (Out_of_Time_Ordered_EchoesV2.pdf): Revised version with a concise abstract, expanded introduction, and additional content, extending to 34 pages.

Both versions cover the same key topics but V2 incorporates refinements for clarity, structure, and depth based on feedback or further development.

Changelog (V1 to V2)

Abstract Refinement: Trimmed for brevity while retaining essential themes. Removed some detailed examples (e.g., specific mentions of CMB/EEG fits) to focus on high-level unification.
Introduction Expansion: Added ~800 words, including a dedicated subsection on objectives, scope, and monograph structure. Enhanced discussion of QFTN's fractal scaling law and its universal application (e.g., CMB fluctuations to EEG coherence).
Content Depth: Minor expansions in sections 3-7, adding derivations or examples (e.g., more explicit connections to Pellis' fractal decoherence [4]). This accounts for the +4 page increase.
Conclusion: Extended to synthesize additions, with stronger emphasis on interdisciplinary implications (e.g., neural echoes, cosmic signatures).
No Major Removals: V2 builds upon V1 without deleting core content; it's an iterative improvement for precision and accessibility.
References: Unchanged, citing key works like Google Quantum AI [1], QFTN papers [2-3], and polyadic SUSY [11].
